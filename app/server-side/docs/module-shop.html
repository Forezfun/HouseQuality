FilterQueryObject

module:findmodule:find

Объект фильтров, где ключ — это поле(ключ field) из Filter, а значение:

Тип фильтра Тип значения
select OptionSelectQuery
range OptionRangeQuery
Type:
  • Object.<string, (module:shop.OptionRangeQuery|module:shop.OptionSelectQuery)>
Source:

Examples
Пример объекта фильтрации
const filters = {
  material: {
    type: 'select',
    value: ['ткань', 'велюр']
  },
  height: {
    type: 'range',
    min: 50,
    max: 120
  }
};
Что передаётся в запрос — JSON.stringify(query)
"{\"material\":{\"type\":\"select\",\"value\":[\"ткань\",\"велюр\"]},\"height\":{\"type\":\"range\",\"min\":50,\"max\":120}}"
